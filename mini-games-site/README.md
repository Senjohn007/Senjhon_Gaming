# Mini Games Site

## Overview
This project is a mini-games site that currently features a "Roll Dice" game. The site is structured with HTML, styled with CSS, and powered by JavaScript for game logic.

## Project Structure
```
mini-games-site
├── index.html       # Basic HTML structure for the site
├── style.css        # CSS styles for the site
├── main.js          # JavaScript for game functionality
└── README.md        # Documentation for the project
```

## Getting Started

### Prerequisites
- A web browser to run the site.
- A code editor (optional) for modifying the files.

### Installation
1. Clone the repository or download the project files.
2. Navigate to the project folder.

### Running the Site
1. Open `index.html` in your web browser.
2. You should see the site with the title and the "Roll Dice" game listed.

## Future Development
- Additional games can be added to the navigation section.
- Enhance the styling in `style.css` for a better user experience.
- Expand the functionality in `main.js` to include more game logic.
